教学AI助手

This project is initialized with 教学AI助手. Follow is the quick guide for how to use.

Environment Prepare

Install node_modules:

npm install

or

yarn

Provided Scripts

教学AI助手 provides some useful script to help you quick start and build with web project, code style check and test.

Scripts provided in package.json. It's safe to modify or add additional script:

Start project

npm start

Build project

npm run build

Check code style

npm run lint

You can also use script to auto fix some lint error:

npm run lint:fix

Test code

npm test

build


docker build -t fe-container:first .
